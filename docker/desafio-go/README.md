# Desafio Go - Full Cycle 3.0

Este repositório contém a solução para a atividade do Desafio Go.

## Descrição do Desafio

Criar uma imagem Docker que imprime "Full Cycle Rocks!!" quando executada, com a restrição de que a imagem final deve ter menos de 2MB de tamanho.

## Imagem Docker

A imagem Docker foi publicada e está disponível em:

```bash
docker run pedronr03/fullcycle
```

**Saída:** `Full Cycle Rocks!!`

## Tamanho da Imagem

O tamanho final da imagem é de **1.5MB**, atendendo aos requisitos do desafio.

## Tecnologias Utilizadas

- Go (Golang)
- Docker
- Multi-stage builds
- Imagem base Scratch
